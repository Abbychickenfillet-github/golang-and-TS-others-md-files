## 15:11 — 傅立葉轉換是什麼

**Claude:**

# 傅立葉轉換（Fourier Transform）

## 一句話解釋

**把訊號從「時間域」轉換成「頻率域」的數學工具** —— 也就是把一段複雜的波形拆解成許多不同頻率的簡單正弦波（sin / cos）的組合。

## 核心想法

法國數學家 Fourier 發現：**任何週期性訊號，都可以表示為無限多個不同頻率正弦波的疊加**。

舉例：你聽到一段交響樂的聲音波形看起來雜亂無章，但其實是小提琴（高頻）+ 鋼琴（中頻）+ 大提琴（低頻）等不同頻率聲音的混合。傅立葉轉換就是「拆解」這段聲音，告訴你：
- 哪些頻率存在？
- 每個頻率的強度有多大？

## 視覺化

```
時間域 (Time Domain)              頻率域 (Frequency Domain)
   雜亂的波形                      清晰的頻率分布

    /\    /\                       │
   /  \  /  \      Fourier        ││
  /    \/    \    ───────►        │││
                                   ┴┴┴───►
   時間 →                          100Hz 200Hz 300Hz
```

## 兩種主要形式

| 名稱 | 用途 |
|------|------|
| **連續傅立葉轉換 (FT)** | 處理連續訊號（理論用） |
| **離散傅立葉轉換 (DFT)** | 處理離散資料（實際工程） |
| **快速傅立葉轉換 (FFT)** | DFT 的高效演算法，O(n log n) |

## 在 NumPy 中怎麼用

```python
import numpy as np

# 假設有一段訊號（50Hz + 120Hz 兩個頻率混合）
t = np.linspace(0, 1, 1000)
signal = np.sin(2*np.pi*50*t) + 0.5*np.sin(2*np.pi*120*t)

# FFT 轉換到頻率域
freq_domain = np.fft.fft(signal)
freqs = np.fft.fftfreq(len(t), d=t[1]-t[0])

# 你會看到 50Hz 和 120Hz 兩個峰值
```

## 實務應用

| 領域 | 應用 |
|------|------|
| 音訊處理 | MP3 壓縮、降噪、等化器（EQ） |
| 影像處理 | JPEG 壓縮、邊緣偵測、模糊濾波 |
| 通訊 | Wi-Fi、4G/5G 訊號調變（OFDM） |
| 醫學影像 | MRI、CT 掃描的影像重建 |
| 地震 / 振動分析 | 找出主要振動頻率 |
| 金融 | 時間序列的週期性分析 |

## 17:52 — 這邊y = A × sin(2π × f × t + φ)求出來的值可以代表什麼

**Claude:**

這是**正弦波（sinusoidal wave）的標準方程式**，每個參數和 `y` 的物理意義取決於應用場景。

## 各參數含義

| 符號 | 名稱 | 意義 |
|------|------|------|
| `A` | 振幅（Amplitude） | 波的最大偏移量 |
| `f` | 頻率（Frequency） | 每秒振動次數，單位 Hz |
| `t` | 時間（Time） | 當前時刻，單位秒 |
| `φ` | 相位（Phase） | 起始角度偏移 |
| `2π × f` | 角頻率 ω | 每秒走過的弧度數 |
| `y` | **瞬時值** | 在時間 `t` 時的當下狀態 |

## `y` 在不同情境代表什麼

### 1. 聲波 / 音訊
`y` = **空氣壓力的瞬時變化**（或喇叭振膜位移、麥克風電壓）
- `A` 大 → 聲音大
- `f` 大 → 音調高（440 Hz = 標準 A4 音）

### 2. 交流電（AC）
`y` = **瞬時電壓或電流**
- 台灣家用電：`A ≈ 156V`、`f = 60Hz`
- `y = 156 × sin(2π × 60 × t)`

### 3. 機械振動
`y` = **物體偏離平衡點的位移**
- 彈簧、單擺、引擎振動

### 4. 電磁波 / 光波
`y` = **電場或磁場強度**

### 5. 訊號處理（DSP）
`y` = **取樣點的數值**（例如 WAV 檔每個 sample）

### 6. 週期性現象
潮汐高度、心跳、呼吸、季節溫度變化等

## 一個直觀理解

把它想成「**一個東西在某時刻離中心有多遠**」：
- 中心點 = 0
- 最高點 = `+A`
- 最低點 = `-A`
- 走完一圈的時間 = `1/f`（週期 T）
- `φ` 決定「從圓的哪個位置開始走」
