## 💻 主要工作內容

### 1. ProtectedRoute 組件改進

#### 支援陣列權限（OR 邏輯）
```typescript
interface ProtectedRouteProps {
  // 可以是單一權限或權限陣列
  permissionRequired: string | string[]
  children: React.ReactNode
}
```

#### 權限不足時顯示 Toast 提示
- 不再只是靜默重定向
- 顯示 5 秒的警告訊息「您無瀏覽本區域的權限」
- 然後重定向到首頁

### 2. 訂單頁面（orders.tsx）權限控制

#### 獨立頁籤權限設計
| 頁籤 | 權限 | 說明 |
|------|------|------|
| 全部 | `orders.view` | 主權限 |
| 票券訂單 | `orders.view` | 主權限 |
| 攤位訂單 | `orders.view` | 主權限 |
| 電力需求 | `orders.view` 或 `order-electricity.view` | 獨立權限 |
| 展場設備訂單 | `orders.view` 或 `order-items.view` | 獨立權限 |
| 退款申請 | `orders.view` 或 `refunds.view` | 獨立權限 |

#### Tab Index 動態計算
修正了當部分 tab 被隱藏時，index 計算錯誤的問題。現在根據可見的 tabs 動態計算 index。

### 3. permissionModules.ts 修改

- 統一權限用語

### 4. RolePermissionTabs.tsx 修改
