## Bug 列表

### Bug 1: 加價購右側顯示錯誤的攤位列表
- **現象**: 加價購模式下，右側顯示的是所有可購買的攤位列表（跟新購一樣）
- **預期**: 右側應該只顯示此人已擁有（已付款）的攤位
- **根因**: `EventRegisterBoothPage.tsx` 右側列表沒有區分 addon/new 模式，一律渲染 `allBooths`
- **檔案**: `src/pages/EventRegisterBoothPage.tsx` line 1225+

### Bug 2: 加價購模式地圖切換消失（多地圖）
- **現象**: 加價購模式下，左側的地圖和地圖切換器被隱藏了
- **預期**: 加價購模式下應該也能看到地圖，並能切換地圖查看不同地圖上的已擁有攤位
- **根因**: line 1004 `{!isAddonMode && (...)}` 直接把地圖藏了
- **檔案**: `src/pages/EventRegisterBoothPage.tsx` line 1004

### Bug 3: 付款出現「訂單不存在」
- **現象**: 進入付款頁面後出現訂單不存在錯誤
- **可能原因**: sessionStorage 的 order draft 遺失或 order_id 沒被正確傳遞
- **待確認**: 需要記錄 sessionStorage 的內容來 debug
- **提醒**: 請記錄 sessionStorage 的完整內容

### Bug 4: 一般購買攤位功能是否正常？
