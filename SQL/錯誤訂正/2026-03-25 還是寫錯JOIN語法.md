# 2026-03-25 JOIN 語法又寫錯

## 我寫錯的
```sql
SELECT * FROM `refund_record` JOIN refund_record.order_id = order.id WHERE o.order_number = "TKT-20260311-1VRLQW"
```

## 三個錯誤

### 1. JOIN 後面要先寫表名，再用 ON 寫條件
- 錯：`JOIN refund_record.order_id = order.id`
- 對：`JOIN \`order\` o ON rr.order_id = o.id`

### 2. 用了 alias `o` 但沒宣告
- `WHERE o.order_number` 的 `o` 從哪來？
- 要在 JOIN 時給別名：`JOIN \`order\` o`

### 3. `order` 是 MySQL 保留字
- 表名用反引號：`` `order` ``
- 或用別名 `o` 之後就不用反引號

## 正確寫法
```sql
SELECT rr.*
FROM refund_record rr
JOIN `order` o ON rr.order_id = o.id
WHERE o.order_number = 'TKT-20260311-1VRLQW'
```

## 記住
- `JOIN 表名 別名 ON 條件` — 不是 `JOIN 條件`
- 反引號只有**表名是保留字**時需要，欄位不用
- 用別名（`rr`, `o`）最省事
