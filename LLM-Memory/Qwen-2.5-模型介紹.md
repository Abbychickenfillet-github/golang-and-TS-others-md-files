# Qwen-2.5(阿里通義千問)模型介紹

## 重點整理

- Qwen-2.5 是阿里雲(Alibaba Cloud)通義千問團隊開發的強大開源語言模型。
- 系列在推理、程式碼、數學等多種任務表現優異,並支援多語言。
- 7B(七十億參數)版本在資源效率與能力之間取得良好平衡。
- 對話中亦提及可進一步瞭解量化(quantization)如何運作。
