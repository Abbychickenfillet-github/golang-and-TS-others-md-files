🔎 10:10 之後，這樣查「今晚改了幾張、進了哪些資料夾」

命名腳本每次跑都會產一份 log：screenshot-rename\logs\rename-YYYYMMDD-HHMMSS.jsonl，裡面有每張圖的原檔名、新檔名、AI 描述。用 PowerShell 這樣統計最快：

# 找出今晚最新的 log
cd C:\coding\futuresign\screenshot-rename
$log = Get-ChildItem logs\rename-*.jsonl | Sort-Object LastWriteTime | Select-Object -Last 1

# 今晚成功改名幾張？
$rows = Get-Content $log.FullName | ForEach-Object { $_ | ConvertFrom-Json }
($rows | Where-Object status -eq ok).Count

# 若有用 --organize-by-month，看分別進了哪些資料夾
$rows | Where-Object status -eq ok |
  Group-Object { Split-Path $_.new -Parent } |
  Select-Object Name, Count | Format-Table -Auto

如果 logs\ 還是空的 → 代表排程今晚還沒跑（或沒有符合「螢幕擷取畫面 ….png」預設名的新圖可改）。
