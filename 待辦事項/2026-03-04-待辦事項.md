## 相關資料表

| 資料表 | 說明 |
|--------|------|
| `map` | 地圖（map_url, original_map_url, config JSON） |
| `map_area` | 地圖區域（name, color, x/y/width/height, is_locked） |
| `booth` | 攤位（map_id, area, event_booth_type_id） |
| `booth_order_subscription` | 攤位訂單關聯 |
| `event_booth_type` | 攤位類型（type_code, display_width/height） |
| `event_booth_type_pricing` | 攤位類型定價（early_bird/standard, target_areas） |

## 規則

- 同一活動的同一張地圖，區域名稱不得重複

- 儲存底圖 = 上傳 S3 + 寫入 map.map_url & map.original_map_url
- 儲存區域 = 寫入 map_area 資料表（非 config JSON）
