## 🎯 面試回答範本

職缺寫 Enzyme，你可以這樣答：

> 「我目前主要用 RTL，但知道 Enzyme 的對應 API —— `shallow` 對應 RTL 的 `render`、`simulate` 對應 `fireEvent` 或 `userEvent`、`find('.btn').text()` 對應 `screen.getByRole('button').textContent`。我認為 React 18 之後 RTL 是合理選擇，因為 Enzyme 跟不上新的並發特性。」

→ **知道 API mapping 比深入 Enzyme 更重要。**
