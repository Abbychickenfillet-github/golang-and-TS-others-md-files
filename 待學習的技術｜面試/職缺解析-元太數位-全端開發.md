# 職缺解析:元太數位 YTDT 全端開發工程師

## 重點整理

把元太數位(YTDT)的全端開發工程師職缺介紹逐句拆解,看穿公司背後的意圖:

### 技術門檻
- 技術棧:**ASP.NET + C# + MSSQL + JS/CSS3/HTML5**(微軟體系)。
- 意圖:要「進來就能接手現有專案」的即戰力;若背景是 React/Node/Go/PostgreSQL,需展現能快速轉換到 .NET 體系。

### 公司背景與護城河
- 2009 年成立,專攻**高等教育(Higher Education)應用系統**——大專院校系統供應商,封閉、高黏著度的利基市場,客戶用了很難換掉。
- 列舉教育部頂尖大學/教學卓越/深耕計畫/十二年國教等 → **吃政府與教育部標案的常勝軍**;產品須高度符合法規、無障礙網頁規範與資安標準(XSS、Injection 防範)。
- 微軟合格供應商、創櫃公司審查、PwC 輔導、金鋒獎 → 強調合規與財務健全,流程嚴謹(可能帶傳統企業階層感)。
- 近年方向:IR 校務研究(數據輔助學校決策)、Data Center 巨量資料、醫院 PGY/UGY 系統(畢業後一般醫學訓練)、K12(幼稚園到高中)線上教學 → 看重**大數據處理與資料庫效能優化**。

### 企業文化與展望
- 「人才培育是關鍵」→ 求穩文化,願意教,但不喜歡高流動率。
- 「過程充滿艱辛」→ 打預防針:新專案(K12/新創)規格變動大,要抗壓性。

### 面試準備 3 大核心方向
1. **資料庫與邏輯能力 > 前端炫技**:強調 Schema design 與複雜商業邏輯、報表正確性。
2. **資安與規範意識**:XSS 防範、權限控管、無障礙網頁,是大加分。
3. **展現轉技術棧的適應力**:「我有良好軟體工程基礎,能快速把邏輯複刻到 C# 與 MSSQL」。

### 待辦
- [ ] 快速瀏覽 C# / ASP.NET 基本語法,理解與熟悉語言的異同
- [ ] 複習 SQL 進階查詢與優化(他們強調 MSSQL 與巨量資料)
- [ ] 把過去專案(活動管理等)的「權限控管」「資料庫設計」經驗連結到教育/醫院系統需求
