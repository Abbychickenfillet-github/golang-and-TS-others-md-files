# NumPy、線性代數與傅立葉轉換（白話版）

## 一、NumPy 是什麼？

**一句話**：NumPy 是 Python 用來處理「一堆數字」的超快工具。

### 為什麼要用它？

Python 原生的 list 處理大量數字很慢，NumPy 提供了 `ndarray`（N 維陣列），運算快上百倍，而且寫法超簡潔。

```python
import numpy as np

# 不用寫 for 迴圈，整組數字一起算
a = np.array([1, 2, 3, 4])
print(a * 2)        # [2, 4, 6, 8]
print(a + 10)       # [11, 12, 13, 14]
```

幾乎所有 Python 的 AI / 資料科學工具（Pandas、TensorFlow、PyTorch）底層都是 NumPy。

## 二、線性代數是什麼？（白話）

線性代數就是在玩「數字組成的箱子」：

| 名稱 | 想像 | 例子 |
|------|------|------|
| 純量 Scalar | 一顆球 | `5` |
| 向量 Vector | 一排球 | `[1, 2, 3]` |
| 矩陣 Matrix | 一個棋盤格 | `[[1, 2], [3, 4]]` |
| 張量 Tensor | 一疊棋盤 | 3D 以上的資料 |

### 為什麼這麼重要？

因為**所有 AI、影像、聲音、推薦系統，最後都是在算矩陣**。學會線性代數 = 拿到 AI 世界的鑰匙。

## 三、最常用的幾個運算

### 1. 向量加法（同位置相加）

```python
a = np.array([1, 2, 3])
b = np.array([4, 5, 6])
a + b   # [5, 7, 9]
```

### 2. 內積 / 點積（兩向量相似程度）

把兩個向量「對應位置相乘再加總」：

```python
a @ b   # 1*4 + 2*5 + 3*6 = 32
```

**用途**：判斷兩個東西像不像。
例如：你的喜好向量 vs 一部電影的特徵向量 → 結果越大表示越合你胃口。

### 3. 矩陣乘法

可以想成「一次做很多次內積」。

```python
A @ B
```

**規則**：左邊矩陣的「欄數」要等於右邊矩陣的「列數」。

`(2,3) × (3,4) = (2,4)` ✅
`(2,3) × (2,4) = ❌`

### 4. 轉置（行欄交換）

```python
A.T
```

把直的變橫的、橫的變直的。

### 5. 反矩陣（除法的概念）

如果 `A × B = 1`，那 B 就是 A 的反矩陣 `A⁻¹`。

```python
np.linalg.inv(A)
```

**用途**：解方程式 `Ax = b` → `x = A⁻¹ b`。

## 四、正弦波（Sine Wave）是什麼？

### 一句話

**最乾淨、最純粹的波**，長得像海浪。

```
    /‾\          /‾\
   /   \        /   \
──/─────\──────/─────\──── 時間軸
         \    /       \
          \__/         \__
```

### 數學公式

```
y = A × sin(2π × f × t + φ)
```

| 符號 | 名稱 | 控制什麼 |
|------|------|----------|
| A | 振幅（Amplitude）| 波多高（聲音多大） |
| f | 頻率（Frequency）| 一秒振幾下（聲音多高） |
| t | 時間 | x 軸 |
| φ | 相位（Phase）| 波從哪個位置開始 |

### NumPy 畫一個正弦波

```python
import numpy as np

t = np.linspace(0, 1, 1000)         # 0 到 1 秒，取 1000 個時間點
wave = np.sin(2 * np.pi * 5 * t)    # 5 Hz 的正弦波（一秒振 5 下）
```

### 為什麼正弦波重要？

1. **聲音**：純音（鋼琴的單一個音）就是正弦波
2. **電力**：插座的交流電是 60Hz 正弦波
3. **無線電**：Wi-Fi、5G 的訊號都是正弦波組合
4. **核心定理**：**任何複雜的波，都可以拆解成很多個正弦波相加** ← 這就是傅立葉轉換在做的事

## 五、傅立葉轉換（白話版）

### 想像一個三稜鏡

- 白光（看起來雜亂）→ 三稜鏡 → 變成彩虹（紅橙黃綠藍靛紫）
- 訊號（看起來雜亂）→ 傅立葉 → 變成各個「頻率成分」

### 例子

你錄一段音樂，波形看起來亂七八糟。傅立葉轉換會告訴你：
- 100Hz 有多大聲（低音）
- 1000Hz 有多大聲（中音）
- 5000Hz 有多大聲（高音）

### NumPy 實作

```python
import numpy as np

# 自己合成一個訊號：50Hz + 120Hz 兩種頻率
t = np.linspace(0, 1, 1000)
signal = np.sin(2*np.pi*50*t) + 0.5*np.sin(2*np.pi*120*t)

# FFT 拆解
result = np.fft.fft(signal)
amplitude = np.abs(result)

# 你會在 50Hz 和 120Hz 看到兩個明顯的峰
```

### 用在哪裡？

- **MP3、JPEG**：把不重要的頻率丟掉 → 檔案變小
- **降噪**：把雜訊頻率過濾掉
- **EQ 等化器**：調整高音、低音
- **MRI 醫學影像**：影像重建

## 六、應用領域白話版

### 機器學習：梯度下降 + 權重更新

#### 梯度下降（Gradient Descent）= 下山找最低點

想像你在山上、眼睛被矇住，怎麼下山？
**摸看哪邊比較斜，往下坡走一小步，再摸一次、再走** → 最後會走到山谷。

AI 訓練時，「山的高度」= 模型的錯誤。AI 不斷調整自己，讓錯誤越來越小。

```
錯誤
 │\
 │ \
 │  \___
 │      \___       ← 一步一步往下走
 │          \__
 │             \____ ← 找到最低點
 └──────────────────► 模型參數
```

#### 權重更新

AI 模型裡面有一堆「旋鈕」叫做**權重（weights）**，每個旋鈕控制一個特徵的重要性。
訓練 = 一直轉旋鈕，直到輸出最準。

例子：判斷垃圾郵件
- 看到「免費」→ 權重 +0.8（很可疑）
- 看到「會議」→ 權重 -0.3（沒事）

#### PCA 降維（Principal Component Analysis）

**問題**：資料有 1000 個欄位太多了，能不能壓成幾個就好？

**PCA 做的事**：找出資料中**最重要的幾個方向**，其他不重要的扔掉。

生活比喻：
- 一個人有身高、體重、腰圍、肩寬、腿長…10 個指標
- PCA 可能告訴你：其實只要看「整體大小」和「身材比例」兩個方向就夠了

```python
# scikit-learn 用法
from sklearn.decomposition import PCA
pca = PCA(n_components=2)   # 壓成 2 維
result = pca.fit_transform(data)
```

### 深度學習：前向 / 反向傳播

神經網路長這樣：

```
輸入 → [第1層] → [第2層] → [第3層] → 輸出
        權重W1    權重W2    權重W3
```

#### 前向傳播（Forward Propagation）

資料**從左往右走**，每經過一層就做一次矩陣乘法：

```
第1層輸出 = 輸入 @ W1
第2層輸出 = 第1層輸出 @ W2
最終輸出 = 第2層輸出 @ W3
```

→ 整個神經網路就是**一連串的矩陣乘法**！

#### 反向傳播（Back Propagation）

發現答錯了怎麼辦？

把錯誤**從右往左傳回去**，告訴每一層的權重：「你錯多少，你要改多少」。
這個過程也是矩陣運算。

**結論**：神經網路 = 矩陣乘法 + 梯度下降。所以線性代數真的很重要。

### 電腦視覺：影像就是矩陣

#### 一張圖 = 一個矩陣

灰階圖：每個像素是一個數字 (0=黑, 255=白)

```
[[ 50, 100, 200],
 [ 80, 150, 220],
 [ 30,  90, 180]]   ← 3x3 的灰階圖
```

彩色圖 = 三個矩陣（R、G、B 三個通道）。

#### 旋轉、縮放都是矩陣乘法

旋轉 90 度、放大 2 倍、變形 → 這些都是把每個像素的座標 `(x, y)` 乘上一個矩陣得到新座標。

```python
# 縮放矩陣
scale = np.array([[2, 0],
                  [0, 2]])

# 把點 (x, y) 變成 (2x, 2y)
new_point = scale @ np.array([x, y])
```

旋轉 θ 度的矩陣：

```python
rotate = np.array([[np.cos(θ), -np.sin(θ)],
                   [np.sin(θ),  np.cos(θ)]])
```

→ 看到沒？又是線性代數又是正弦波。

### 推薦系統：SVD 矩陣分解 + 協同過濾

#### 場景

Netflix 有 100 萬個用戶 × 10 萬部電影，組成一個超大評分矩陣：

```
            電影A  電影B  電影C  電影D
用戶1        5      ?      3      ?
用戶2        ?      4      ?      2
用戶3        4      ?      5      ?
```

大部分格子是空的（沒看過），怎麼**猜測用戶會給沒看過的電影幾分**？

#### SVD（奇異值分解）

把這個大矩陣**拆成三個小矩陣相乘**：

```
評分矩陣 ≈ 用戶特徵矩陣 × 重要性 × 電影特徵矩陣
   M     ≈     U        ×   S    ×    V
```

意義：
- **U**：每個用戶喜歡什麼類型（動作？愛情？）
- **V**：每部電影屬於什麼類型
- 把它們乘回去 → **可以把空格也填出來**！

這就是「協同過濾」的原理：用其他用戶的行為，預測你會喜歡什麼。

```python
U, S, V = np.linalg.svd(matrix)
```
