# 副檔名圖鑑 🗂️

> 記錄遇到的「奇怪、少見」副檔名的意義與用途,遇到新的就往下加。
> 用途:看到陌生檔案時知道它是幹嘛的、能不能刪/能不能編輯。

## 判斷原則

- **能不能刪**:系統/程式設定檔、資料庫檔 → 別亂刪;快取、log、暫存 → 通常可刪。
- **能不能編輯**:純文字類(.json/.yml/.ini/.env/.md)可以;二進位類(.dll/.exe/.vscdb/.db)不要用記事本硬改,會壞。

---

## 🗄️ 資料庫 / 儲存

| 副檔名 | 意義 | 備註 |
| --- | --- | --- |
| `.vscdb` | VS Code / 其分支(Cursor、Antigravity)的 **SQLite 狀態資料庫** | 存工作區狀態、聊天紀錄等;二進位,別手改 |
| `.db` / `.sqlite` / `.sqlite3` | **SQLite 資料庫**檔 | 一個檔就是一個資料庫 |
| `.wal` | SQLite 的 **Write-Ahead Log**(尚未寫回主檔的變更) | 例:`DIPS-wal`;跟著主 db 一起,別單獨刪 |
| `.dat` | 通用資料檔 | 例 `NTUSER.DAT` = 使用者登錄檔(**不可刪**) |
| `.vhdx` / `.vhd` | **虛擬硬碟**(檔案假裝成一顆磁碟) | Docker/WSL/VM 用;vhdx 是新一代(見 [[Docker問題prune+壓縮vhdx(虛擬磁碟)解決硬碟爆掉]]) |

## ⚙️ 程式 / 系統(二進位,別手改)

| 副檔名 | 意義 | 備註 |
| --- | --- | --- |
| `.exe` | 可執行程式 | Windows 應用主體 |
| `.dll` | **動態連結函式庫**(Dynamic Link Library) | 多個程式共用的功能模組;缺了程式會報「找不到 xxx.dll」 |
| `.sys` | Windows **系統驅動/核心**檔 | 絕對別碰 |
| `.msi` | Windows **安裝封裝**(Installer) | 雙擊即安裝 |
| `.msc` | **Microsoft 管理主控台**(MMC snap-in) | 例 `diskmgmt.msc`(磁碟管理)、`eventvwr.msc`(事件檢視器)、`services.msc` |
| `.lnk` | **捷徑**檔 | 指向別的檔/程式 |
| `.vsix` | **VS Code 擴充套件**安裝包 | |

## 📜 腳本 / 設定(純文字,可編輯)

| 副檔名 | 意義 | 備註 |
| --- | --- | --- |
| `.bat` / `.cmd` | Windows **批次檔**(CMD 腳本) | 注意 BOM 問題會害它閃退 |
| `.ps1` | **PowerShell** 腳本 | 預設被執行原則擋,需設定 |
| `.sh` | **Shell** 腳本(Linux/Mac) | |
| `.json` | 結構化資料 / 設定 | 到處都是 |
| `.yml` / `.yaml` | 設定檔(縮排敏感) | docker-compose、CI 常用 |
| `.toml` | 設定檔 | Python `pyproject.toml`、Rust 常用 |
| `.ini` / `.cfg` | 舊式設定檔 | |
| `.env` | **環境變數**檔 | 常放密鑰,別 commit 進 git |
| `.lock` | **鎖定/版本鎖**檔 | 例 `code.lock`、`package-lock.json` |
| `.log` | 純文字日誌 | 通常可刪 |

## 🐍 程式語言產物

| 副檔名 | 意義 |
| --- | --- |
| `.pyc` | Python 編譯後的位元組碼(快取,可刪) |
| `.whl` | Python **wheel** 套件安裝包 |

## 📝 待補(遇到新的往這加)

| 副檔名 | 意義 | 日期 |
| --- | --- | --- |
| `.updaterId` | App 自動更新器用的**安裝識別碼**(裡面是一串 ID),讓更新機制認得這台裝置的這份安裝。二進位/文字皆有,通常很小(幾十 bytes)。別刪,刪了 App 可能重生或影響更新。 | 2026-07-12 |
| `.lock`(補充) | **鎖定檔**:程式用它宣告「我正在跑/正在用這個資源」,防止多個實例同時開、或同時寫入衝突。例:`code.lock` = 編輯器正在執行。App 正常關閉會自動釋放;若當機殘留變「殭屍鎖」,確認 App 已關再刪即可。 | 2026-07-12 |
