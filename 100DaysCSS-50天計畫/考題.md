## ✅ Checkpoint A/B — 基礎圖形與盒模型（Day 1-10）

**A. 觀念選擇/是非**
1. (是非) `box-sizing: border-box` 會把 `padding` 和 `border` 算進元素的 `width` 內。
2. (選擇) 要把一個 `div` 變成正圓形，需要？ (a) `border-radius: 10px` (b) 寬高相等 + `border-radius: 50%` (c) `border-radius: 100px`
3. (是非) `position: absolute` 的元素是相對「最近一個有 `position` 非 static 的祖先」定位。

**B. 看 CSS 說結果**
```css
.box { width: 100px; padding: 20px; border: 5px solid; box-sizing: border-box; }
```
這個 `.box` 實際內容區（content）寬度是多少？

**C. 易錯陷阱**
```css
.parent { /* 沒寫 position */ }
.child { position: absolute; top: 0; left: 0; }
```
`.child` 會貼在 `.parent` 左上角嗎？為什麼？

**D. 手寫實作**
用一個 `div` 畫出「在畫面正中央的紅色圓形」，直徑 100px。

## ✅ Checkpoint C/D — Flexbox 與元件排版（Day 11-20）

**A. 觀念選擇/是非**
1. (是非) `justify-content` 控制的是 flex「主軸」方向的對齊。
2. (選擇) 預設 `flex-direction: row` 時，`align-items: center` 會讓子元素？ (a) 水平置中 (b) 垂直置中 (c) 兩者都置中
3. (是非) `gap` 可以用在 flex 容器來設定子元素間距。

**B. 看 CSS 說結果**
```css
.container {
  display: flex;
  justify-content: space-between;
  align-items: center;
  height: 200px;
}
```
三個子元素會怎麼排（水平/垂直）？

**C. 易錯陷阱**
你想用 flex 讓單一元素「水平垂直置中」，寫了 `justify-content: center` 但忘了另一個。少了哪一行？只設一個會怎樣？

**D. 手寫實作**
用 flexbox 做一個「左邊 logo、右邊選單」的導覽列（兩端對齊、垂直置中）。

## ✅ Checkpoint E/F — transition 與 hover（Day 21-30）

**A. 觀念選擇/是非**
1. (是非) `transition` 要寫在「平常狀態」的選擇器上，hover 才會有進有出（雙向過渡）。
2. (選擇) `transition: all .3s ease` 的 `.3s` 是？ (a) 延遲 (b) 持續時間 (c) 重複次數
3. (是非) 改 `transform: scale()` 比改 `width/height` 更適合做動畫，因為前者通常不觸發 reflow，效能較好。

**B. 看 CSS 說結果**
```css
.btn { transform: scale(1); transition: transform .2s; }
.btn:hover { transform: scale(1.2); }
```
滑鼠移上去、再移開，分別會發生什麼？有沒有過渡動畫？

**C. 易錯陷阱**
```css
.btn { background: blue; }
.btn:hover { background: red; transition: background .3s; }
```
為什麼「移上去變紅有動畫，但移開瞬間跳回藍色、沒動畫」？怎麼修？

**D. 手寫實作**
做一個按鈕：hover 時放大 1.1 倍、背景由藍變綠，過渡 0.25 秒。

## ✅ Checkpoint G/H — keyframes 動畫（Day 31-40）

**A. 觀念選擇/是非**
1. (是非) `@keyframes` 裡的 `0%` 等同 `from`、`100%` 等同 `to`。
2. (選擇) 要讓動畫無限循環，要加？ (a) `animation-fill-mode: forwards` (b) `animation-iteration-count: infinite` (c) `animation-delay: infinite`
3. (是非) `animation-delay` 可以是負值，效果是「動畫從中間某個進度開始」。

**B. 看 CSS 說結果**
```css
@keyframes spin { to { transform: rotate(360deg); } }
.loader { animation: spin 1s linear infinite; }
```
`.loader` 會怎麼動？`linear` 代表什麼？

**C. 易錯陷阱**
你做了一個跑完就「彈回原狀」的動畫，想讓它「停在結束狀態」。少了哪個屬性？該設成什麼？

**D. 手寫實作**
寫一個 loading 小圓點：用 `@keyframes` 讓它在 0%、50%、100% 之間上下跳動（`translateY`），無限循環。

## ✅ Checkpoint I/J — 進階組合（Day 41-50）

**A. 觀念選擇/是非**
1. (是非) 一個元素可以同時套用多個 `animation`，用逗號分隔。
2. (選擇) 要做 3D 翻牌效果，父層通常要設？ (a) `overflow: hidden` (b) `perspective` (c) `z-index`
3. (是非) `clip-path` 可以把元素裁切成三角形、多邊形等形狀。

**B. 看 CSS 說結果**
```css
.card { transform: rotateY(180deg); backface-visibility: hidden; }
```
這張卡片正面看得到嗎？`backface-visibility: hidden` 的作用是？

**C. 易錯陷阱**
你用 `linear-gradient` 當背景但「看不到漸層、整片同色」。最常見的兩個原因是什麼？

**D. 手寫實作**
用 `clip-path` 把一個正方形 `div` 裁成一個三角形（指向上方）。

## Checkpoint A/B
**A.** 1. 是。 2. (b)。 3. 是。
**B.** content 寬 = 100 − 20×2(padding) − 5×2(border) = **50px**（因為 border-box 把 padding/border 算進 100px 內）。
**C.** 會貼在 `.parent`，但**只有當 `.parent` 有設 `position`（relative 等）**。題目 `.parent` 沒設 position，所以 `.child` 會往上找有 position 的祖先，最後貼到 `<body>/viewport`，不是 `.parent`。修法：`.parent { position: relative; }`。
**D.**
```css
.circle {
  width: 100px; height: 100px;
  background: red; border-radius: 50%;
  position: absolute; top: 50%; left: 50%;
  transform: translate(-50%, -50%);
}
```

## Checkpoint C/D
**A.** 1. 是。 2. (b) 垂直置中（row 時交叉軸是垂直）。 3. 是。
**B.** 三個子元素**水平排列**、彼此間「兩端對齊、平均留白」，且在 200px 高度內**垂直置中**。
**C.** 少了 `align-items: center`。只設 `justify-content: center` 只會水平置中，垂直仍貼上緣。要兩者都設才真正置中。
**D.**
```css
.nav {
  display: flex;
  justify-content: space-between;
  align-items: center;
}
```

## Checkpoint E/F
**A.** 1. 是。 2. (b) 持續時間。 3. 是。
**B.** 移上去：在 0.2s 內平滑放大到 1.2 倍；移開：平滑縮回 1。兩個方向都有過渡，因為 `transition` 寫在平常狀態。
**C.** 因為 `transition` 寫在 `:hover` 上，只有「進入 hover」那一刻存在；移開時 hover 規則消失，沒有 transition 可套，所以瞬間跳回。修法：把 `transition` 移到 `.btn`（平常狀態）。
**D.**
```css
.btn { background: blue; transition: all .25s; }
.btn:hover { background: green; transform: scale(1.1); }
```

## Checkpoint G/H
**A.** 1. 是。 2. (b)。 3. 是。
**B.** 持續旋轉，每 1 秒轉一圈、永不停止。`linear` = 等速（沒有加減速）。
**C.** 少了 `animation-fill-mode: forwards`（讓動畫停在最後一幀的狀態）。
**D.**
```css
@keyframes bounce {
  0%, 100% { transform: translateY(0); }
  50%      { transform: translateY(-12px); }
}
.dot { animation: bounce .6s ease-in-out infinite; }
```

## Checkpoint I/J
**A.** 1. 是。 2. (b) `perspective`。 3. 是。
**B.** 看不到正面（被轉到背面 180°）；`backface-visibility: hidden` 讓元素「背面朝向你時不顯示」，常用在翻牌避免看到鏡像背面。
**C.** 兩個常見原因：(1) 兩個色標（color stop）顏色一樣或太接近；(2) 沒給元素「寬高」或漸層被單色背景覆蓋（例如又寫了 `background: #fff` 蓋掉）。
**D.**
```css
.triangle {
  width: 100px; height: 100px; background: teal;
  clip-path: polygon(50% 0, 100% 100%, 0 100%);
}
```
