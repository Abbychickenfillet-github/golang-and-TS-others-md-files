# 100 Days CSS — 50 天衝刺計畫

> 來源：[100dayscss.com](https://100dayscss.com/) ｜ 共 100 個 CSS 臨摹挑戰
> 節奏：**一天 2 題，共 50 天**（每天約 30-60 分鐘）
> 方式：每題拿官方 [CodePen 範本](https://codepen.io/pen?template=wMQKOq&editors=1100)，看著目標設計用純 CSS 還原。
> 搭配 [[考題]]（CSS 觀念題）＋ 同資料夾 `進度追蹤表.xlsx`。

## 🔁 每天的固定流程（兩題都一樣）

1. **看（Observe）**：先盯著當天那題的目標設計，心裡想「這是用什麼排版/技巧做的？」
2. **拆（Plan）**：拆成結構（HTML）＋ 外觀（顏色/圓角/陰影）＋ 動態（transition/animation）。
3. **做（Build）**：用範本動手刻，盡量純 CSS。
4. **比（Compare）**：跟原圖比對，記下「卡住的點」與「新學到的屬性」到自己的筆記。
5. **連（Link）**：遇到的觀念去 [[考題]] 找對應題自測；可在 Obsidian 連到既有 CSS 筆記。

> 進階建議：每完成 10 天（20 題）回頭重看第 1 題，你會明顯感覺到差異。
