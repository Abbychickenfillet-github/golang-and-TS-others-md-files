## Checkpoint 1
**A.** 1. 否（const 鎖的是「綁定」，物件內部仍可改；要全鎖用 `Object.freeze`）。 2. (c) `var`。 3. 是。
**B.** `"123"`（1+"2"→"12"，再+3→"123"）；`3`（"5"-2，減法強制轉數字）；`2`（true→1）。
**C.** `false`。浮點以二進位儲存有精度誤差，`0.1+0.2 = 0.30000000000000004`。正確比較：`Math.abs(0.1+0.2-0.3) < Number.EPSILON`。
**D.** `const toNumber = s => { const n = Number(s); return Number.isNaN(n) ? 0 : n; };`

## Checkpoint 2
**A.** 1. 否，`switch` 用 `===`（嚴格相等）。 2. (c) `"0"`（非空字串是 truthy）。
**B.** 印出 `two` 和 `three`。case 2 命中後沒有 break，會 fall-through 到 case 3 才 break。
**C.** 印出 `01`、`11`、`21`。`for...in` 拿到的是「索引字串」`"0"`、`"1"`、`"2"`，字串 + 數字變字串相接。要拿值用 `for...of`。
**D.**
```js
for (let i = 1; i <= 100; i++) {
  if (i % 7 === 0) continue;
  if (i % 3 === 0) console.log(i);
}
```

## Checkpoint 3
**A.** 1. 是。 2. (b) `["a","b","c"]`。
**B.** `"llo"`（負索引從尾取 3 個）；`"X-b-a-b"`（replace 預設只換第一個，要全換用 `replaceAll` 或正則 `/a/g`）；`15`（parseInt("12px")→12，Number("3")→3）。
**C.** `true`。兩個都是字串，按字典序逐字元比較，`"3"`(51) > `"1"`(49)，第一個字元就分勝負。
**D.** `const reverseString = s => s.split("").reverse().join("");`

## Checkpoint 4
**A.** 1. 否，`forEach` 回傳 `undefined`。 2. (c) `map`（push/splice/sort 都會 mutate 原陣列）。
**B.** `[20, 40]`（先濾偶數 [2,4] 再 ×10）；`10`（1+2+3+4）。
**C.** `[1, 10, 2, 30]`。`sort` 預設把元素轉字串比較，所以 "10" < "2"。正確：`arr.sort((a, b) => a - b)`。
**D.**
```js
const count = [1,2,2,3,3,3].reduce((acc, n) => {
  acc[n] = (acc[n] || 0) + 1;
  return acc;
}, {});
// { 1:1, 2:2, 3:3 }
```

## Checkpoint 5
**A.** 1. 否，會丟 ReferenceError（TDZ 暫時死區）；`var` 才是 undefined。 2. (b) 定義它的外層作用域。
**B.** `undefined`（var a 被 hoist）；然後丟 `ReferenceError: Cannot access 'b' before initialization`。
**C.** `"Hi Abby"` 與 `"Hi undefined"`。一般函式的 `this` 是呼叫它的物件 `obj`；箭頭函式沒有自己的 `this`，沿用外層（這裡是模組/全域），所以 `this.name` 是 undefined。
**D.**
```js
function makeCounter() {
  let count = 0;
  return () => ++count;
}
const c = makeCounter();
c(); // 1  c(); // 2
```

## Checkpoint 6
**A.** 1. 否，`Set` 的值唯一。 2. (b) `"object"`（JS 歷史 bug）。
**B.** `4`（重複值被去掉，剩 1,2,3 → 但 size 是 3？注意：[1,1,2,3,3] 去重後是 {1,2,3}，size = 3）。更正：`3`。第二行 `10 5`（x 用預設值 10，y 取到 5）。
**C.** `99`。`b = a` 複製的是「參考」，兩個變數指向同一個物件，改 b 等於改 a。要獨立要拷貝：`{...a}`。
**D.**
```js
const arr = ["a","b","a","c","b","a"];
const unique = [...new Set(arr)];              // ["a","b","c"]
const counts = arr.reduce((m, v) => m.set(v, (m.get(v)||0)+1), new Map());
// Map { "a"=>3, "b"=>2, "c"=>1 }
```

## 期末卷
**Part A.** 1. 是。 2. 否（map 回傳新陣列、不改原陣列）。 3. (c) `0`（"false"、`[]`、`{}` 都 truthy）。 4. (b) 丟 SyntaxError。 5. 是。
**Part B.**
1. `[4, 6]`（map→[2,4,6]，filter >2 →[4,6]）。
2. `"number" "object" "object"`。
3. `"ReferenceError"`（函式內 `let x` 造成 TDZ，`x = 2` 在宣告前賦值）。
4. `3 "b" "c"`。
**Part C.**
1. `![]` 先把 `[]` 轉布林：`[]` 是 truthy → `![]` = `false`。於是變成 `[] == false`；`==` 兩邊轉數字：`false`→0，`[]`→`""`→0，`0 == 0` → `true`。
2. 因為 `var i` 是函式作用域，三個 setTimeout 共用同一個 `i`，迴圈跑完時 `i` 已是 3。修法：把 `var` 改成 `let`（每輪一個新綁定），或用 IIFE 包住。
**Part D.**
```js
// 1
function groupBy(arr, key) {
  return arr.reduce((acc, item) => {
    const k = item[key];
    (acc[k] ||= []).push(item);
    return acc;
  }, {});
}
// 2
const isPalindrome = s => {
  const t = s.toLowerCase();
  return t === t.split("").reverse().join("");
};
// 3
class Stack {
  #items = [];
  push(x) { this.#items.push(x); }
  pop() { return this.#items.pop(); }
  peek() { return this.#items[this.#items.length - 1]; }
  get size() { return this.#items.length; }
}
```
