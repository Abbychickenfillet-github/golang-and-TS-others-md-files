# JS 2 週密集複習計畫 (W3Schools)

> 程度設定：**有基礎想複習** ｜ 期程：**14 天**，每天約 2-3 小時
> 來源：[W3Schools JavaScript Tutorial](https://www.w3schools.com/js/default.asp)
> 搭配 [[考題]] 一起用：每幾天有 Checkpoint 小考，第 14 天總複習考。

## 📅 每天怎麼用這份計畫

每天分三段，建議比例約 4 : 4 : 2：

1. **讀 (Read)**：把當天主題的 W3Schools 頁面讀過，重點記到自己的筆記。
2. **練 (Practice)**：每頁的 **「Try it Yourself »」** 一定要自己改一次，再做當天的「練習任務」。
3. **檢 (Check)**：做當天的自我檢核 / Checkpoint 題（見 [[考題]]），錯的回頭補。

> 勾選框可以在 Obsidian 直接點。Excel 版進度表見同資料夾 `進度追蹤表.xlsx`。

## 第 1 週：語法核心 + 流程控制

### Day 1 — 基礎與變數
- [ ] [JS Introduction](https://www.w3schools.com/js/js_intro.asp)
- [ ] [JS Where To](https://www.w3schools.com/js/js_whereto.asp)
- [ ] [JS Output](https://www.w3schools.com/js/js_output.asp)
- [ ] [JS Syntax](https://www.w3schools.com/js/js_syntax.asp) / [Statements](https://www.w3schools.com/js/js_statements.asp) / [Comments](https://www.w3schools.com/js/js_comments.asp)
- [ ] [JS Variables](https://www.w3schools.com/js/js_variables.asp) / [Let](https://www.w3schools.com/js/js_let.asp) / [Const](https://www.w3schools.com/js/js_const.asp)
- [ ] [JS Types](https://www.w3schools.com/js/js_types.asp)

**練習任務**：用 `var` / `let` / `const` 各宣告一個變數，故意製造一個「重新賦值 const 報錯」的例子並看錯誤訊息。
**複習重點**：`var` vs `let` vs `const` 差在哪（作用域、可否重新賦值、hoisting）。

### Day 2 — 運算子
- [ ] [JS Operators](https://www.w3schools.com/js/js_operators.asp)
- [ ] [JS Arithmetic](https://www.w3schools.com/js/js_arithmetic.asp)
- [ ] [JS Assignment](https://www.w3schools.com/js/js_assignment.asp)
- [ ] [JS Comparisons](https://www.w3schools.com/js/js_comparisons.asp)
- [ ] [JS Type Conversion](https://www.w3schools.com/js/js_type_conversion.asp)

**練習任務**：寫出 5 組會讓你猶豫的比較，例如 `0 == ""`、`null == undefined`、`"5" + 3`、`"5" - 3`、`[] == false`，先猜結果再用 console 驗證。
**複習重點**：`==` vs `===`（型別轉換）、`+` 在字串/數字下的差異。✅ Checkpoint 1（見考題）

### Day 3 — 條件判斷
- [ ] [JS If / If Else](https://www.w3schools.com/js/js_if_else.asp)
- [ ] [JS Ternary](https://www.w3schools.com/js/js_if_ternary.asp)
- [ ] [JS Switch](https://www.w3schools.com/js/js_switch.asp)
- [ ] [JS Booleans](https://www.w3schools.com/js/js_booleans.asp)
- [ ] [JS Logical (&& \|\| !)](https://www.w3schools.com/js/js_logical.asp)

**練習任務**：把一段 if-else 改寫成 switch，再改寫成三元運算子；故意漏掉 switch 的 `break` 看 fall-through 效果。
**複習重點**：switch 的 `break` 與 `default`、truthy/falsy 值清單。

### Day 4 — 迴圈
- [ ] [JS Loops](https://www.w3schools.com/js/js_loops.asp)
- [ ] [JS Loop for](https://www.w3schools.com/js/js_loop_for.asp)
- [ ] [JS Loop while](https://www.w3schools.com/js/js_loop_while.asp)
- [ ] [JS Break](https://www.w3schools.com/js/js_break.asp) / [Continue](https://www.w3schools.com/js/js_continue.asp)
- [ ] [JS Control Flow](https://www.w3schools.com/js/js_control_flow.asp)

**練習任務**：用 for、while、`for...of`、`for...in` 各印一次同一個陣列，觀察 `for...in` 拿到的是 index 還是 value。
**複習重點**：`break` vs `continue`、`for...of`（值）vs `for...in`（鍵）。✅ Checkpoint 2

### Day 5 — 字串
- [ ] [JS Strings](https://www.w3schools.com/js/js_strings.asp)
- [ ] [JS String Templates](https://www.w3schools.com/js/js_string_templates.asp)
- [ ] [JS String Methods](https://www.w3schools.com/js/js_string_methods.asp)
- [ ] [JS String Search](https://www.w3schools.com/js/js_string_search.asp)

**練習任務**：用樣板字串 `` `Hello ${name}` `` 組句子；練 `slice` / `substring` / `replace` / `split` / `includes` / `indexOf`。
**複習重點**：字串不可變 (immutable)、`slice` 負索引、`replace` 預設只換第一個。

### Day 6 — 數字與 Math
- [ ] [JS Numbers](https://www.w3schools.com/js/js_numbers.asp) / [Number Methods](https://www.w3schools.com/js/js_number_methods.asp)
- [ ] [JS Number Properties](https://www.w3schools.com/js/js_number_properties.asp)
- [ ] [JS Bitwise](https://www.w3schools.com/js/js_bitwise.asp)（略讀）/ [BigInt](https://www.w3schools.com/js/js_bigint.asp)（略讀）
- [ ] [JS Math](https://www.w3schools.com/js/js_math.asp) / [Math Random](https://www.w3schools.com/js/js_random.asp)

**練習任務**：寫一個產生 1~100 隨機整數的函式；練 `parseInt` / `parseFloat` / `Number()` / `toFixed`；測 `0.1 + 0.2 === 0.3`。
**複習重點**：浮點誤差、`NaN`、`parseInt` 與 `Number()` 差別。✅ Checkpoint 3

### Day 7 — 函式（重點日）
- [ ] [Function Intro](https://www.w3schools.com/js/js_function_intro.asp) / [Invocation](https://www.w3schools.com/js/js_function_invocation.asp)
- [ ] [Parameters](https://www.w3schools.com/js/js_function_parameters.asp) / [Returns](https://www.w3schools.com/js/js_function_return.asp) / [Arguments](https://www.w3schools.com/js/js_function_arguments.asp)
- [ ] [Function Expressions](https://www.w3schools.com/js/js_function_expressions.asp)
- [ ] [Arrow Functions](https://www.w3schools.com/js/js_arrow_function.asp)

**練習任務**：同一個功能用「函式宣告」「函式表達式」「箭頭函式」各寫一次；練預設參數與 rest 參數 `(...args)`。
**複習重點**：函式宣告 vs 表達式（hoisting 差異）、箭頭函式的 `this` 與一般函式不同。✅ 第 1 週總 Checkpoint

## 第 2 週：資料結構 + 物件導向 + 進階

### Day 8 — 陣列（基礎）
- [ ] [JS Arrays](https://www.w3schools.com/js/js_arrays.asp)
- [ ] [JS Array Methods](https://www.w3schools.com/js/js_array_methods.asp)
- [ ] [JS Array Const](https://www.w3schools.com/js/js_array_const.asp)

**練習任務**：練 `push` / `pop` / `shift` / `unshift` / `splice` / `slice` / `concat`，並比較 `splice`（改原陣列）vs `slice`（不改）。
**複習重點**：哪些方法會「改到原陣列 (mutate)」、哪些回傳新陣列。

### Day 9 — 陣列（高階方法，重點日）
- [ ] [JS Array Search](https://www.w3schools.com/js/js_array_search.asp)
- [ ] [JS Array Sort](https://www.w3schools.com/js/js_array_sort.asp)
- [ ] [JS Array Iterations](https://www.w3schools.com/js/js_array_iteration.asp)

**練習任務**：對一個數字陣列做 `map`（每個 ×2）、`filter`（取偶數）、`reduce`（加總）、`find` / `findIndex`；用 `sort((a,b)=>a-b)` 正確排數字。
**複習重點**：`sort` 預設按「字串」排（`[10,2,1]` 會錯）、`map` 回傳新陣列、`forEach` 沒回傳值。✅ Checkpoint 4（MDN：[Array.prototype.map](https://developer.mozilla.org/zh-TW/docs/Web/JavaScript/Reference/Global_Objects/Array/map)）

### Day 10 — 物件
- [ ] [Object Intro](https://www.w3schools.com/js/js_object_intro.asp) / [Properties](https://www.w3schools.com/js/js_object_properties.asp)
- [ ] [Object Methods](https://www.w3schools.com/js/js_object_methods.asp) / [Object this](https://www.w3schools.com/js/js_object_this.asp)
- [ ] [Object Constructors](https://www.w3schools.com/js/js_object_constructors.asp) / [Display](https://www.w3schools.com/js/js_object_display.asp)

**練習任務**：建一個 `person` 物件含屬性與方法；用 `Object.keys` / `Object.values` / `Object.entries` 走訪；練解構 `const {name, age} = person`。
**複習重點**：物件方法裡的 `this`、點記法 vs 中括號記法。

### Day 11 — 作用域與 Hoisting
- [ ] [JS Scope](https://www.w3schools.com/js/js_scope.asp)
- [ ] [JS Code Blocks](https://www.w3schools.com/js/js_codeblocks.asp)
- [ ] [JS Hoisting](https://www.w3schools.com/js/js_hoisting.asp)
- [ ] [JS Strict Mode](https://www.w3schools.com/js/js_strict.asp)

**練習任務**：寫一段在宣告前就使用 `var` 與 `let` 的程式，比較「`undefined`」與「ReferenceError (TDZ)」；測試區塊作用域。
**複習重點**：函式作用域 vs 區塊作用域、hoisting 對 var/let/const/function 的不同行為。✅ Checkpoint 5

### Day 12 — Set / Map / 迭代器
- [ ] [JS Sets](https://www.w3schools.com/js/js_sets.asp) / [Set Methods](https://www.w3schools.com/js/js_set_methods.asp)
- [ ] [JS Maps](https://www.w3schools.com/js/js_maps.asp) / [Map Methods](https://www.w3schools.com/js/js_map_methods.asp)
- [ ] [JS Iterables](https://www.w3schools.com/js/js_iterables.asp) / [Iterators](https://www.w3schools.com/js/js_iterators.asp)（略讀 [Generators](https://www.w3schools.com/js/js_generators.asp)）

**練習任務**：用 `Set` 去除陣列重複值；用 `Map` 存 key-value 並用 `for...of` 走訪；比較 `Map` 與一般物件當字典的差異。
**複習重點**：`Set` 值唯一、`Map` 的 key 可以是任意型別、兩者都用 `.size` 不是 `.length`。

### Day 13 — 資料型別 / 日期 / RegExp
- [ ] [JS Data Types](https://www.w3schools.com/js/js_datatypes.asp) / [Primitives](https://www.w3schools.com/js/js_datatypes_primitives.asp) / [typeof](https://www.w3schools.com/js/js_typeof.asp)
- [ ] [JS Destructuring](https://www.w3schools.com/js/js_destructuring.asp)
- [ ] [JS Dates](https://www.w3schools.com/js/js_dates.asp) / [Date Methods](https://www.w3schools.com/js/js_date_methods.asp)
- [ ] [JS RegExp](https://www.w3schools.com/js/js_regexp.asp)（略讀，懂基本 pattern 即可）

**練習任務**：用 `typeof` 測 7 種值，注意 `typeof null === "object"` 這個經典陷阱；練物件/陣列解構與預設值。
**複習重點**：基本型別 vs 物件型別、`typeof null` 的歷史 bug、淺拷貝 vs 深拷貝概念。✅ Checkpoint 6

### Day 14 — OOP / JSON / 錯誤處理 + 總複習
- [ ] [JS Classes](https://www.w3schools.com/js/js_classes.asp) / [Inheritance](https://www.w3schools.com/js/js_class_inheritance.asp) / [Static](https://www.w3schools.com/js/js_class_static.asp)
- [ ] [JS JSON](https://www.w3schools.com/js/js_json.asp)
- [ ] [JS Errors (try/catch/throw)](https://www.w3schools.com/js/js_errors.asp)
- [ ] [JS Debugging](https://www.w3schools.com/js/js_debugging.asp)（含 [Console](https://www.w3schools.com/js/js_debugging_console.asp)）

**練習任務**：寫一個 `class` 含 constructor、method、繼承 `extends`；用 `JSON.stringify` / `JSON.parse` 來回轉換；用 `try...catch` 包一段會丟錯的程式。
**複習重點**：class 是語法糖、`JSON.parse` 失敗會丟錯、`try/catch/finally` 流程。

## ✅ 收尾自我檢核

讀完 14 天後，問自己能不能不看資料說清楚：

1. `var` / `let` / `const` 三者差異 + hoisting 行為？
2. `==` vs `===`、truthy/falsy 有哪些？
3. `map` / `filter` / `reduce` / `forEach` 各回傳什麼、會不會改原陣列？
4. 箭頭函式的 `this` 跟一般函式差在哪？
5. `typeof null` 是什麼？為什麼？
6. `JSON.parse` 失敗會發生什麼、怎麼處理？

六題都能講清楚，就代表這輪複習到位了。
