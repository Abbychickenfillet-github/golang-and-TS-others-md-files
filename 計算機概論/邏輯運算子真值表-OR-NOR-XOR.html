邏輯運算子真值表:OR / NOR / XOR

重點整理

真值表只是把 A、B 兩個條件的所有組合列出來,看清運算結果。把 A、B 想成兩個問題,運算子就是「要不要採取行動」的判斷方式。

OR(或)

只要任一個條件為 True,結果就是 True;只有 A、B 兩個「全部都是 False」時,OR 才會是 False。

NOR(或非)

NOR 是 OR 的相反(NOT OR)。把整個 OR 的結果反轉:

  • 只要有任一個是 True → OR 為 True → NOR 為 False。
  • 只有「兩個都是 False」時,OR 為 False → NOR 才是 True。

[!warning] 常見誤解:NOR「沒有 False 才是 False」嗎?不對。正確理解是「OR 反過來」——兩個都 True 時,OR=True,所以 NOR=False;唯有兩個都 False,NOR 才 True。

XOR(互斥或 Exclusive OR)

規則:A、B 中「只有一個」是 True,結果才是 True;兩個都 True、或兩個都 False(即兩者相同)時,結果為 False。可記成「兩者不同 → True;兩者相同 → False」。

[!tip] 一句話記憶:OR =「至少一個真」;NOR =「全部都假」才真;XOR =「恰好一個真 / 兩者相異」才真。
