進制轉換與「除 2 取餘數法」原理

重點整理

為什麼「一直除以 2、餘數倒過來」就是二進制?

把十進制數字 N 轉成二進制時,常用「除 2 取餘數法」:不斷除以 2、記下每次餘數,最後把餘數由下往上(從最後一個餘數讀回第一個)排列就是答案。

關鍵疑問:為什麼第一個餘數是最低位元(LSB)、最後一個餘數是最高位元(MSB)?

原理來自多項式除法(Polynomial Division)。任何十進制數字 N 都可以寫成以 2 為底的多項式:

N = aₙ·(2ⁿ) + … + a₂·(2²) + a₁·(2¹) + a₀·(2⁰)

其中每個係數 aᵢ 只會是 0 或 1,就是二進制的各個位元。

當你把這個多項式除以 2 時:

  • 含有 2¹、2²、…、2ⁿ 的項都能被 2 整除 → 變成商。
  • 只有最低次的 a₀·(2⁰) = a₀ 無法被整除 → 它就是餘數。

所以第一次相除的餘數,必定對應 2⁰ 這一項,也就是最低位元;接著對商繼續除 2,下一個餘數對應 2¹…依此類推。等除到商為 0,最後一個餘數對應最高次幂,就是最高位元。這就是「餘數要倒過來讀」的數學根據。

多項式除法基礎

不論整數或多項式除法,都符合同一個基本關係:

被除式 = 除式 × 商式 + 餘式
f(x) = g(x)·q(x) + r(x)

重要規則(次數限制):餘式 r(x) 的最高次數一定要小於除式 g(x) 的最高次數。例如除式是二次方(x²),餘式最多只能是一次方(ax+b)或常數;餘式為 0 代表「整除」。

兩種主要計算法:

方法 適用 特點
長除法 Long Division 任何多項式相除 最通用、最直覺;降冪排列,缺項補 0,逐步消去最高次項
綜合除法 Synthetic Division 除式為一次式 (x − c) 只抓係數做加減乘,計算量大幅減少,適合快速求因式或求值

[!tip] 一句話總結:十進制→二進制的「除 2 取餘」其實就是把 N 看成以 2 為底的多項式,反覆做多項式除法、一層層把係數(位元)從低位到高位剝出來。任何進制轉任何進制都是同理(除以目標底數即可)。
