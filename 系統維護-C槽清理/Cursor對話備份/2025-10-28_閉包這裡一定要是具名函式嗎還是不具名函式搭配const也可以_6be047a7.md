## 解答

**不需要，閉包不一定要具名函式。** 具名、匿名或箭頭函式皆可用。

### 三種寫法都可以實作閉包：

1. **具名函式**（原本的寫法）：
   ```javascript
   function dogHouse(){
       var count=0;
       function countDogs(){
           ++count;
           return count + 'dogs';
       }
       return countDogs
   }
   ```

2. **不具名函式 + const**（第 112-119 行）：
   ```javascript
   const dogHouse2 = function(){
       var count=0;
       return function(){
           ++count;
           return count + 'dogs';
       }
   }
   ```

3. **箭頭函式**（第 121-128 行）：
   ```javascript
   const dogHouse3 = () => {
       var count=0;
       return () => {
           ++count;
           return count + 'dogs';
       }
   }
   ```

### 重點

閉包與命名方式無關，而是：
- 外層函式作用域持有變數
- 內層函式存取該變數
- 內層函式持續持有該變數的存取

開啟瀏覽器開發者工具，可在 console 看到上述寫法都能正確計數。
