## Cursor
這個 `GlitchText` 組件的傳入參數使用了 **解構賦值 (destructuring assignment)** 語法，讓我詳細解釋：

## 參數說明

```javascript
const GlitchText = ({ children, className = '' }) => {
```

### 1. `children`
- **類型**: React 節點 (ReactNode)
- **用途**: 要顯示在 glitch 效果中的文字內容
- **範例**: `<GlitchText>Hello World</GlitchText>` 中的 "Hello World" 就是 children

### 2. `className = ''`
- **類型**: 字串 (string)
- **預設值**: 空字串 `''`
- **用途**: 額外的 CSS 類別名稱，用於自訂樣式
- **範例**: `<GlitchText className="my-custom-class">Text</GlitchText>`

## 使用方式

```javascript
// 基本使用
<GlitchText>Hello World</GlitchText>

// 帶自訂樣式
<GlitchText className="text-center">Welcome</GlitchText>

// 組合使用
<GlitchText className="text-lg font-bold">
  Glitch Effect Text
</GlitchText>
```

## 組件內部運作

1. **第6行**: 將 `children` 同時作為顯示內容和 `data-text` 屬性
2. **第6行**: 組合預設樣式 `styles.text` 和使用者傳入的 `className`
3. **第7行**: 渲染 `children` 內容

這個組件設計得很靈活，可以接受任何文字內容並套用 glitch 效果，同時允許使用者透過 `className` 參數添加額外的樣式。
