### 什麼是 Babel？

**Babel** 是一個 JavaScript 編譯器，主要功能是：

1. **語法轉換**：將現代 JavaScript 語法（ES6+）轉換成舊版瀏覽器能理解的語法
2. **Polyfill 注入**：為舊瀏覽器添加缺失的功能
3. **JSX 轉換**：將 React 的 JSX 語法轉換成 JavaScript
4. **TypeScript 支援**：將 TypeScript 轉換成 JavaScript

### 在你的專案中

```json
"devDependencies": {
  "@babel/preset-env": "^7.26.0",
  "@babel/preset-react": "^7.25.9",

}
```
