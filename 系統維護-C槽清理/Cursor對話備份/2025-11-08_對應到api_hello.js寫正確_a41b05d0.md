更完整的寫法範例：

```javascript
export default function handler(req, res) {
  // 只處理 GET 請求
  if (req.method === 'GET') {
    res.status(200).json({ name: "John Doe" });
  } else {
    // 其他方法返回 405 Method Not Allowed
    res.status(405).json({ error: 'Method not allowed' });
  }
}
```

**為什麼這樣回傳？**
- `res.status(200)`：HTTP 狀態碼 200 表示成功
- `.json()`：將 JavaScript 物件轉為 JSON 字串，並設定正確的 `Content-Type` header
- `{ name: "John Doe" }`：回傳的資料結構，前端可用 `data.name` 取得
