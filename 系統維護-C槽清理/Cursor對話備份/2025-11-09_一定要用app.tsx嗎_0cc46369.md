## 不同架構的主入口檔案慣例

### 1. **Vite + React**
- 主入口：`main.tsx` 或 `main.jsx`（在 `index.html` 中引用）
- 根組件：通常叫 `App.tsx`（可選，非必須）
- 結構：
  ```
  index.html → main.tsx → App.tsx → 其他組件
  ```

### 2. **Next.js (App Router)**
- 主入口：`app/layout.tsx`（根布局，自動作為入口）
- 頁面：`app/page.tsx`（首頁）
- 沒有傳統的 `main.tsx` 或 `App.tsx`
- 結構：
  ```
  app/layout.tsx (根布局) → app/page.tsx (頁面)
  ```

### 3. **Next.js (Pages Router)**
- 主入口：`pages/_app.tsx`（應用入口）
- 頁面：`pages/index.tsx`（首頁）
- 結構：
  ```
  pages/_app.tsx → pages/index.tsx
  ```

### 4. **Create React App (CRA)**
- 主入口：`src/index.tsx` 或 `src/index.js`
- 根組件：`src/App.tsx`
- 結構：
  ```
  index.tsx → App.tsx
  ```

### 5. **Remix**
- 主入口：`app/root.tsx`（根組件）
- 路由：`app/routes/` 下的檔案
