**為什麼需要小括號的原因：**

1. **語法要求**：箭頭函數的簡寫形式必須用小括號 `()` 包圍返回值
2. **避免歧義**：沒有小括號的話，JavaScript 會將 `=>` 後面的內容視為函數體，而不是返回值
3. **明確表達意圖**：小括號讓程式碼更清楚地表達這是一個返回表達式的函數

**兩種寫法的對比：**
- 完整寫法：`() => { return expression }`
- 簡寫寫法：`() => (expression)` 或 `() => expression`（單行時可省略小括號）

在這個例子中，由於是多行，所以必須使用小括號來包圍整個表達式。
