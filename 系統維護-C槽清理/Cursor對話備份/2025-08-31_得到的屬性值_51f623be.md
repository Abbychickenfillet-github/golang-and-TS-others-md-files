## Cursor
不一定。那只是函式的參數名稱，你可以叫任何名字。

- 直接接參數：
```jsx
function Child(xxx) {
  return <div>{xxx.title}</div>
}
```

- 建議用解構（更常見）：
```jsx
function Child({ title, price }) {
  return <div>{title} - {price}</div>
}
```

重點：React 只會把父母透過 JSX 傳入的屬性物件當作第一個參數傳進來，參數名稱不影響行為。
