### 需要使用哪种终端机？

在 Windows 上有三种选择：

#### 选项 1: Git Bash（推荐，最简单）

如果你已经安装了 Git for Windows：
- 右键点击项目目录 → 选择 "Git Bash Here"
- 直接运行 `.sh` 脚本，无需额外设置

#### 选项 2: PowerShell（Windows 原生）

使用 Windows 内建的 PowerShell：
- 需要先设置执行策略（一次性设置）
- 使用 `.ps1` 脚本

#### 选项 3: WSL（如果你有）

如果你使用 WSL，可以直接使用 Linux 版本的脚本
