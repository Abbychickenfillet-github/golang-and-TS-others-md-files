## Cursor
**Prettier 設定：**
- `.prettierrc.json` 設為 `"singleQuote": true`，應使用單引號

**為何 `settings.json` 使用雙引號：**
- JSON 標準只允許雙引號，Prettier 不會把 JSON 的雙引號改成單引號
- 因此 `.vscode/settings.json` 與 `.prettierrc.json` 本身都使用雙引號

**在程式碼中：**
- JavaScript/TypeScript 會使用單引號（依 `.prettierrc.json`）
- JSON 檔案會使用雙引號

**驗證：**
```typescript
// 這會使用單引號（符合 Prettier 設定）
const message = 'Hello world'
```

```json
// 這會使用雙引號（JSON 標準）
{
  "singleQuote": true
}
```

結論：設定正確。JSON 用雙引號，程式碼用單引號。
