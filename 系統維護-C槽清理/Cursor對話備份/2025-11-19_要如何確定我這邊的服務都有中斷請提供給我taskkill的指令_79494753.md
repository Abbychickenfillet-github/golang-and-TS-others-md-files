## 快速终止指令

### 方法 1: 通过端口终止进程

```powershell
# 终止占用端口 8000 的进程 (FastAPI)
netstat -ano | findstr :8000
# 然后使用返回的 PID，例如：
taskkill /F /PID <PID>

# 终止占用端口 4444 的进程 (Vite)
netstat -ano | findstr :4444
# 然后使用返回的 PID，例如：
taskkill /F /PID <PID>
```

### 方法 2: 一键终止所有相关进程

```cmd
# 终止所有 Python 进程（包括 uvicorn）
taskkill /F /IM python.exe

# 终止所有 Node.js 进程（包括 Vite）
taskkill /F /IM node.exe
```

### 方法 3: 使用我创建的脚本

**批处理文件 (推荐):**
```cmd
cd be-fastapi
stop_all_services.bat
```

**PowerShell 脚本:**
```powershell
cd be-fastapi
.\stop_all_services.ps1
```

### 方法 4: 检查端口是否被占用

```cmd
# 检查端口 8000
netstat -ano | findstr :8000

# 检查端口 4444
netstat -ano | findstr :4444
```

## 完整的 PowerShell 一行指令

```powershell
# 终止端口 8000 的进程
Get-NetTCPConnection -LocalPort 8000 -ErrorAction SilentlyContinue | ForEach-Object { Stop-Process -Id $_.OwningProcess -Force }

# 终止端口 4444 的进程
Get-NetTCPConnection -LocalPort 4444 -ErrorAction SilentlyContinue | ForEach-Object { Stop-Process -Id $_.OwningProcess -Force }
```

建议使用 `stop_all_services.bat` 或 `stop_all_services.ps1`，它们会自动检查并终止所有相关服务，并显示结果。
