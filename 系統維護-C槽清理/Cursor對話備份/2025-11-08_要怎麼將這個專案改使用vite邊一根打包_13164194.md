## Cursor
是的，Vite 和 Next.js 通常是互斥的。说明如下：

- Next.js：全栈 React 框架，自带构建系统（Webpack/Turbopack），提供路由、SSR、API 路由等。
- Vite：构建工具，专注于快速开发体验和构建，不提供路由或 SSR。

选择：
- 使用 Next.js：用 Next.js 的构建系统，获得其框架功能。
- 使用 Vite：用 Vite 构建纯 React 应用，需要自己处理路由等。
