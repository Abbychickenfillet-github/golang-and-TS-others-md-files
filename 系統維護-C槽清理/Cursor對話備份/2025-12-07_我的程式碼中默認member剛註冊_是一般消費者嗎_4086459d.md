## Cursor
`"ui.dim"?:"inherit"?` 不是有效的 TypeScript/JavaScript 語法。目前程式碼使用的是三元運算符，可簡化如下：

## 當前寫法（正確）

```typescript
color={!company.company_name ? "ui.dim" : "inherit"}
```

## 可能的簡化方式

### 1. 反轉邏輯（更直觀）
```typescript
color={company.company_name ? "inherit" : "ui.dim"}
```

### 2. 使用邏輯或運算符（但需注意空字串）
```typescript
// 注意：如果 company_name 是空字串 ""，這會返回 "ui.dim"
color={company.company_name || "ui.dim" ? "inherit" : "ui.dim"}
// 這其實沒有簡化，不推薦
```

### 3. 使用空值合併運算符（僅適用於 null/undefined）
```typescript
// 只對 null/undefined 有效，空字串不會觸發
color={company.company_name ?? "ui.dim" ? "inherit" : "ui.dim"}
// 這也沒有簡化
```

## 建議

保持現有的三元運算符寫法，或反轉邏輯以提高可讀性：

```typescript
// 原寫法
color={!company.company_name ? "ui.dim" : "inherit"}

// 反轉邏輯（更直觀）
color={company.company_name ? "inherit" : "ui.dim"}
```

兩者功能相同，後者更直觀。

## Cursor
`ui.dim` 的顏色是 **`#A0AEC0`**（灰藍色）。

在主題定義中：

```28:28:frontend/src/theme.tsx
      dim: "#A0AEC0",
```

這是一個中性的灰藍色，常用於：
- 顯示未填寫或缺失的資料（如 "N/A"）
- 次要文字或提示文字
- 表示內容不完整或需要補充的狀態

在程式碼中，當欄位為空時會使用 `ui.dim` 讓文字變淡，表示該欄位缺少資料。
