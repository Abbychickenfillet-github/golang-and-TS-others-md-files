## 快速说明

### 三个双引号 `"""` 的作用

1. 创建多行字符串
   ```python
   text = """第一行
   第二行
   第三行"""
   ```

2. 文档字符串（Docstring）
   - 放在函数/类的第一行
   - 用于说明功能、参数、返回值
   - IDE 和工具会读取并显示

### 在你的代码中的例子

```python
def create_organizer(...):
    """创建主办方资料"""  # ← 这是文档字符串
    # 函数实现...
```

### 多行字符串 vs 单行字符串

```python
# 单行字符串
str1 = "Hello World"

# 多行字符串（使用三个双引号）
str2 = """第一行
第二行
第三行"""

# 等价于
str3 = "第一行\n第二行\n第三行"
```

### 在 FastAPI 中的重要性

文档字符串会被自动用于生成 API 文档：
- Swagger UI (`/docs`)
- ReDoc (`/redoc`)
- IDE 的代码提示

已创建详细指南文档：`be-fastapi/PYTHON_STRING_GUIDE.md`，包含更多示例和最佳实践。
