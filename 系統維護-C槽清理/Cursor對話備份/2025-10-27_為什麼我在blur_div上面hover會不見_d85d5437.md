# 為什麼我在blur div上面hover會不見

> Cursor · 2025-10-27 18:31 · `d85d5437-52bd-4501-ad13-540ff14e46df` · 4 則

## 你
為什麼我在blur div上面hover會不見？

## Cursor
檢查 `blur` div 的樣式以找出問題：

## Cursor
hover 時看起來消失，原因是強烈的模糊：

```20:32:drop-shadow.html
.blur {
    width: 200px;
    height: 200px;
    background-color: #f0f0f0;  // 你目前的檔案顯示是 #962b2b
    border-radius: 5px;
}

.blur:hover {
    filter: blur(150px);  // ← 這裡是問題！150px 是極度誇張的模糊
    margin: 5px;
    width: 300px;
    height: 300px;
}
```

原因：
1. blur(150px) 過於強烈：內容被發散超過元素範圍，變得肉眼不可見
2. 尺寸突變：200px → 300px 可能造成移位感
3. 背景色與 `body` 相同（#f0f0f0），會融為一體

處理方式：降低模糊強度（例如改為 blur(10px)），必要時加上顯眼的背景色以避免視覺消失。

## Cursor
已調整：

**修改內容：**
- 模糊度：從 `blur(150px)` 降至 `blur(5px)`
- 尺寸：從 `300px` 降為 `210px`，hover 變化更自然

hover 時不再消失，可見模糊漸變效果。若要更明顯，可依需求再微調數值。
