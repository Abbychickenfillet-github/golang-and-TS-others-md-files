## Cursor
`jsconfig.json` 是 JavaScript 專案的設定檔，用於：

## 主要功能

### 1. **路徑別名 (Path Mapping)**
```json
"paths": {
  "@/*": ["./*"]
}
```
- 設定 `@/` 指向專案根目錄
- 可用 `@/components/Button` 取代 `../../../components/Button`

### 2. **基礎路徑 (Base URL)**
```json
"baseUrl": "."
```
- 設定模組解析的基礎路徑
- 讓 import 路徑更簡潔

### 3. **排除目錄 (Exclude)**
```json
"exclude": ["node_modules", "dist", ".next", "build"]
```
- 告訴 IDE 不要掃描這些目錄
- 提升效能，減少不必要的警告

## 實際效果

**沒有 jsconfig.json 時：**
```javascript
import Button from '../../../components/Button'
import { api } from '../../../../services/api'
```

**有 jsconfig.json 後：**
```javascript
import Button from '@/components/Button'
import { api } from '@/services/api'
```

## 與 TypeScript 的關係

- `jsconfig.json` = JavaScript 專案用
- `tsconfig.json` = TypeScript 專案用
- 兩者功能類似，但 `jsconfig.json` 專為 JS 專案設計

## 其他好處

1. 更好的 IDE 支援：自動完成、跳轉定義
2. 更快的編譯：排除不必要的檔案
3. 更清晰的專案結構：統一的路徑解析

總結：`jsconfig.json` 讓 JavaScript 專案的路徑管理更簡潔、IDE 支援更好。

## 1. `noValidate` 和 `needs-validation` 是什麼？

### `noValidate`
```javascript
<form noValidate>
```
- HTML5 屬性，關閉瀏覽器原生表單驗證
- 避免瀏覽器預設驗證訊息
- 改用自訂驗證邏輯

### `needs-validation`
```javascript
<form className="needs-validation">
```
- Bootstrap 5 的 CSS 類別
- 啟用 Bootstrap 自訂驗證樣式
- 與 `noValidate` 搭配使用

## 2. 為什麼註冊的輸入框會有聚焦效果？

聚焦效果來自 Bootstrap 的 `form-control` 類別：

```css
.form-control:focus {
  border-color: #86b7fe;
  outline: 0;
  box-shadow: 0 0 0 0.25rem rgba(13, 110, 253, 0.25);
}
```

- 聚焦時邊框變藍
- 出現藍色陰影
- 移除預設 outline

## 3. CSS 類別說明

### `fw-semibold` (Label)
```javascript
<label className="text-white fw-semibold">
```
- Bootstrap 字重類別
- `fw-semibold` = `font-weight: 600`
- 讓標籤更醒目

### `form-control` (Input)
```javascript
<input className="form-control form-control-lg">
```
- Bootstrap 表單控制項樣式
- 提供基本樣式、聚焦效果、響應式

### `form-control-lg` (Input)
```javascript
<input className="form-control form-control-lg">
```
- Bootstrap 大尺寸
- 增加 padding 與字體大小
- 提升可點擊區域

## 4. 完整的樣式組合效果

```javascript
// Label: 白色文字 + 半粗體
<label className="text-white fw-semibold">

// Input: 大尺寸表單控制項 + 聚焦效果
<input className="form-control form-control-lg">
```

效果：
- 標籤：白色、半粗體
- 輸入框：大尺寸、聚焦藍色邊框與陰影
- 表單：使用自訂驗證，不使用瀏覽器原生驗證
