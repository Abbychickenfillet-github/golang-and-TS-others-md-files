## 你
padding有三個參數的話分別是哪邊

## Cursor
在 CSS 的 `padding` 屬性中，當有三個參數時，它們分別代表：

## CSS Padding 三參數語法：

```css
padding: top right bottom;
```

### 具體對應：
1. **第一個參數**：`top` - 上邊距
2. **第二個參數**：`right` - 右邊距  
3. **第三個參數**：`bottom` - 下邊距
4. **左邊距**：自動等於右邊距（`left = right`）

## 在你的 Footer 中的例子：

```scss
.footer {
  padding: 2rem 0 1rem;
  //     ↑   ↑   ↑
  //     │   │   └── bottom: 1rem (下邊距)
  //     │   └────── right: 0 (右邊距)
  //     └────────── top: 2rem (上邊距)
  //     left: 0 (自動等於 right)
}
```

### 所以你的 Footer 實際上是：
- **上邊距**：`2rem`
- **右邊距**：`0`
- **下邊距**：`1rem`
- **左邊距**：`0`（自動等於右邊距）

## CSS Padding 參數規則：

| 參數數量 | 語法 | 說明 |
|---------|------|------|
| 1個 | `padding: 10px` | 四個方向都是 10px |
| 2個 | `padding: 10px 20px` | 上下 10px，左右 20px |
| 3個 | `padding: 10px 20px 30px` | 上 10px，左右 20px，下 30px |
| 4個 | `padding: 10px 20px 30px 40px` | 上 10px，右 20px，下 30px，左 40px |

記憶口訣：**順時針方向** - 上 → 右 → 下 → 左
