# dict 搭配中括號 `[]` 的存取寫法

## 為什麼 dict 用 `{}` 又用 `[]`？

dict 在三種情境下會出現「括號」，但用的是**不同種類**：

| 情境 | 括號 | 範例 |
|------|------|------|
| 定義字面值 | `{}` 大括號 | `d = {"a": 1, "b": 2}` |
| 存取 / 設定 / 刪除 | `[]` 中括號 | `d["a"]`、`d["c"] = 3`、`del d["a"]` |
| 型別註記 | `[]` 中括號 | `d: dict[str, int]` |

這篇專門講「中括號存取」這一種。

---

## 1. 取值（讀）

```python
inventory = {"sku-001": "攤位 A", "sku-002": "攤位 B"}

print(inventory["sku-001"])   # 攤位 A
```

### key 不存在會炸：`KeyError`

```python
inventory["sku-999"]
# KeyError: 'sku-999'
```

不想炸，要用 [`.get()` 方法](../dict-get方法.md)：

```python
inventory.get("sku-999")              # None（不炸）
inventory.get("sku-999", "無此商品")   # 預設值
```

**口訣**：

- 確定 key 存在 → 用 `[]`（快、直接）
- 不確定 key 是否存在 → 用 `.get()`（安全）

---

## 2. 設值（寫）

```python
inventory["sku-003"] = "攤位 C"      # 不存在 → 新增
inventory["sku-001"] = "攤位 A 更新"  # 已存在 → 覆蓋
```

dict 的 key 是**唯一的**，重複賦值就是覆蓋，不會像 list 一樣新增一筆。

對照 Java：

```java
map.put("sku-003", "攤位 C");   // Java 用方法
```

Python 用中括號賦值，更接近原生語法。

---

## 3. 刪除

```python
del inventory["sku-001"]
```

key 不存在也會炸 `KeyError`，要先檢查：

```python
if "sku-001" in inventory:
    del inventory["sku-001"]

# 或用 .pop() 帶預設值，不炸
inventory.pop("sku-999", None)
```

---

## 4. 巢狀 dict 的中括號鏈

```python
inventory = {
    "sku-001": {"name": "攤位 A", "qty": 3},
    "sku-002": {"name": "攤位 B", "qty": 1},
}

# 一層一層取
print(inventory["sku-001"]["name"])   # 攤位 A
print(inventory["sku-001"]["qty"])    # 3

# 也可以一層一層改
inventory["sku-001"]["qty"] = 99
```

每對 `[]` 都是一次 dict 存取。如果中間某層 key 不存在，整串就炸。

---

## 5. 中括號的 key 可以是什麼？

dict 的 key 必須是 **hashable（可雜湊）** 的型別，中括號裡塞什麼就照規則來：

```python
# ✅ 字串
inventory["sku-001"]

# ✅ 數字
prices = {1: 100, 2: 200}
prices[1]

# ✅ tuple
coords = {("B", 12): "攤位"}
coords[("B", 12)]      # 注意要連 tuple 一起塞

# ❌ list 不能當 key（unhashable）
bad = {[1, 2]: "X"}    # TypeError
```

---

## 6. 跟 list、Java、JS 對照

| 語言 / 結構 | 取值寫法 | 備註 |
|------|----------|------|
| Python `dict` | `d["key"]` | 不存在 → `KeyError` |
| Python `list` | `lst[0]` | 用整數 index |
| JS `object` | `o["key"]` 或 `o.key` | Python 沒有點記法 |
| JS `Map` | `m.get("key")` | 必須用方法 |
| Java `HashMap` | `m.get("key")` | 必須用方法 |

Python dict 用 `[]` 取值的設計，是把 dict 跟 list 統一在「容器 + 中括號 + index/key」的模型下。

---

## TL;DR

```python
d = {"a": 1}        # {} 定義
d["a"]              # [] 取值 → 1
d["b"] = 2          # [] 設值（新增或覆蓋）
del d["a"]          # [] 刪除
"a" in d            # 檢查 key 存在（不用括號）
d.get("z")          # 不確定 key 存在時用，不炸
```

**核心**：dict 的中括號 `[]` = key 索引存取，跟 list 的 `[]` = 整數索引存取概念一致，差別只在 key 的型別不同。
